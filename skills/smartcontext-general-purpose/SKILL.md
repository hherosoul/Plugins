# SmartContext Skill: General Purpose

## Domain Summary

General-purpose conversation guidelines suitable for all scenarios. No matter what topic you're discussing, these rules will help focus on the most valuable information.

## Priority Declaration

When rules from this domain conflict with other domains, priority: **Medium**

## Priority Guidelines (Tier 0 → Tier 4)

### Tier 0 — Always Prioritize (Never Overlook)
- Content explicitly marked as important by the user
- Core constraints and rules that must be followed
- User's core needs and goals
- Key personal information and preferences provided by the user

### Tier 1 — Highest Priority Focus
- Critical decisions and important choices
- Core business logic and concepts
- Nature of the problem and root causes
- Reached consensus and conclusions

### Tier 2 — High Priority Attention
- Detailed design and implementation details
- List of functional requirements
- Specific examples and explanations
- Background information and context

### Tier 3 — Keep In Mind
- Configuration items and settings
- Debug records and process information
- Minor details and supplementary explanations
- Formatting and style information

### Tier 4 — Can Reference Briefly
- Confirmatory responses ("Okay", "Got it", "Received", etc.)
- Duplicate content (keep only the last one)
- General knowledge (common sense already in LLM)
- Polite phrases and greetings

## Dynamic Adjustment Rules

1. **Citation Frequency**: Information repeatedly mentioned by the user → Promote 1 Tier
2. **Time Decay**: Tiers 3-4 decay over time, Tiers 0-2 do not decay
